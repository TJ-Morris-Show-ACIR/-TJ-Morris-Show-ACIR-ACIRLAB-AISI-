## Dashboard Overview

The Capstone Tracker consolidates progress across all 150 mission-critical projects. Use this page to access sprint metrics,
release readiness indicators, and retrospective notes.

### Key Metrics

- **Delivery Velocity**: Average number of capstones reaching production each sprint.
- **Automation Coverage**: Percentage of capstones leveraging reusable automation components.
- **Telemetry Health**: Number of projects with complete observability instrumentation.

### Reporting Cadence

| Cadence | Audience | Purpose |
|---------|----------|---------|
| Weekly | Mission Control | Highlight blockers and upcoming releases |
| Monthly | Executive Steering | Summarize business impact and automation ROI |
| Quarterly | Cross-Functional Council | Align roadmap and resource allocation |

### How to Update

1. Open the Capstone Tracker workspace in Jet Services Control Center.
2. Update milestone statuses and attach latest artifacts (architecture diagrams, test results, decision records).
3. Tag mission control agents to request governance reviews or AI alignment checks.
4. Publish a summary note outlining next steps and risks.
